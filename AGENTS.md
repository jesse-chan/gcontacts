# Project Instructions

## Browser verification

- 對 Google Cloud Console、GitHub Pages、Search Console 或其他網頁後台進行完成度、狀態與設定檢查時，只要操作是唯讀的，Codex 應主動開啟 Browser，使用目前已登入的工作階段查證。
- 不要先要求使用者提供截圖；只有在 Browser 無法連線、尚未登入、遇到 CAPTCHA，或頁面權限不足時才請使用者介入。
- 唯讀檢查可以直接執行，包括開啟頁面、切換檢視、展開選單與讀取設定。
- 未經使用者明確要求，不得修改、儲存、建立、刪除、發布或提交任何設定。
- 回報時應區分「Browser 已確認」、「repo 已確認」與「仍無法確認」。
